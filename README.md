# IPL-Dashboard
1. Project Title / Headline

IPL 2022 Performance Analysis Dashboard

 2. Short Description / Purpose

An interactive Power BI dashboard built to analyze IPL 2022 match data, focusing on team performance, toss impact, scoring patterns, and venue-based insights to support data-driven analysis.

 3. Tech Stack
Power BI
Power Query (ETL)
DAX (KPIs & Measures)
Excel

4. Data Source
IPL 2022 dataset (Excel file – match-wise data including teams, scores, toss, venue, and players)

 5. Features / Highlights
🔹 Business Problem

Understanding factors affecting match outcomes such as toss decisions, scoring trends, and team performance.

🔹 Key Questions
How does the toss impact match results?
What are the scoring patterns in first vs second innings?
Which teams perform consistently across matches?
How do venues influence match outcomes?

🔹 Dashboard Highlights
KPIs for first innings and second innings scores
Team-wise performance analysis
Toss decision impact visualization
Interactive filters (Match ID, Toss Winner, Player of the Match)
Venue-based map visualization
